## ✍️ Writing Guidelines

### 1. **Natural Keyword Integration**
❌ **Bad**: "desk setup desk setup office setup wfh setup"
✅ **Good**: "Whether you need a WFH setup, office desk setup, or dream workspace, GSpaces delivers complete desk setup solutions."

### 2. **Keyword Density**
- Use primary keywords 2-3 times per 100 words
- Use variations naturally throughout content
- Don't force keywords - write for humans first

### 3. **Content Structure**
```
H1: Include primary keyword (e.g., "Premium Desk Setup Solutions")
H2: Include secondary keywords (e.g., "WFH Desk Setup Ideas")
H3: Include long-tail keywords (e.g., "Ergonomic Home Office Setup")
```

## 🖼️ Image Optimization

### Alt Tag Format:
```html
<!-- Bad -->
<img src="desk.jpg" alt="desk">

<!-- Good -->
<img src="executive-desk-setup.jpg" alt="GSpaces executive desk setup with ergonomic chair and modern lighting for home office">
```

### Image Naming:
- Use descriptive names: `gspaces-wfh-desk-setup-minimalist.jpg`
- Include keywords: `ergonomic-office-desk-setup-india.jpg`
- Avoid generic names: `IMG_1234.jpg`

## ✅ SEO Best Practices

1. **Content Length**: Aim for 1000+ words for blog posts, 300+ for product descriptions
2. **Keyword Placement**: Include primary keyword in first 100 words
3. **Headings**: Use H1, H2, H3 with keywords
4. **Images**: Always add descriptive alt tags
5. **Links**: Include 3-5 internal links per page
6. **Mobile**: Ensure content is mobile-friendly
7. **Speed**: Optimize images (compress to <200KB)
8. **Updates**: Refresh content every 3-6 months
