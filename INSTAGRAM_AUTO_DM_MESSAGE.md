## Implementation Notes

### Instagram DM Character Limits
- Instagram DMs support up to **1,000 characters**
- All versions above are within this limit
- Choose based on your brand voice and audience

### Best Practices
1. **Timing**: Send DM within 5-10 minutes of user engagement
2. **Personalization**: Consider adding user's first name if available
3. **Link**: Use a shortened URL if needed (bit.ly, tinyurl)
4. **Follow-up**: Set up automated reminder after 24 hours if not claimed

### Technical Requirements
- Ensure coupon code `GSPACES_DESKS_FOLLOW` is active in your database
- Verify wallet system is properly configured
- Test the entire flow before going live

### Compliance
- ✅ Include opt-out option if sending multiple messages
- ✅ Follow Instagram's automation policies
- ✅ Don't spam users with repeated messages
- ✅ Respect user privacy and data protection laws
